# TP-GestionTrafic


## Technologie
* JavaFX


## Lancement

Quand vous lancez le projet, l'utilisateur doit donner le nombre de routes souhaité et le nombre de voiture sur une ligne.
Une fois les informations entrées, une fenetre va apparaitre. Elle représentera le traffic routier avec le nombre de routes et de véhicules demandés ainsi que des feux de circulation à chaque intersection.
